# Doceian Doors US Market Content Plan

## Purpose

This document prepares Phase 4 content improvements for the Doceian Doors static website.

The goal is to strengthen the site for the U.S. interior door market, especially:
- distributors
- builders and contractors
- wholesalers
- millwork suppliers
- project buyers

This is a planning document. It should not include unsupported claims as publish-ready copy.

- ✓ Edit live pages with safe, verified content

### What Phase 4B Will NOT Do:

- ❌ Add testimonials without consent

- Preserve HTML structure and CSS classes
